# anisotrope-cube

## Requirements
* python 3.5>
* OpenFOAM
* SALOME
* ParaView

## Getting started

* Clone project:
```bash
$ git clone git@github.com:L-Nafaryus/anisotrope-cube.git
$ cd anisotrope-cube
```

* Generate all structures with configured parameters:
```bash
$ python run.py mesh
```

* Run calculations:
```bash
$ python run.py calc
``` 

* All:
```bash
$ python run.py all
```
